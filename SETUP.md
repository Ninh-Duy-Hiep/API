**Các API**
# API lấy danh sách tất cả các thuốc
`http://localhost:3000/api/medicines` 
# API tìm kiếm theo thuốc 
`http://localhost:3000/api/medicines/search?searchTerm=keyword`

`http://localhost:3000/api/medicines` 
# API thêm file thuốc
`http://localhost:3000/api/medicines/upload`
